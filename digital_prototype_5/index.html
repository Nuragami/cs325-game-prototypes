Color Clash

  • High Concept: Fast paced color coordination dodging game.
  • Theme: Black and colored blocks
  • Mandated Variety: Keyboard input
  • Player Experience Goals: The main experience to take away is the ability to test one's coordination under pressure.
  • Gameplay: Left and Right arrow keys to move side ways. Q key to change to normal color, W key to change to blue, E key to change to red, and R key to change to green. The color keys must be held down. Change colors and try to dodge colors that to not match the color of the player. If the block is the same color as the player, it does not hurt them. There are hidden black blocks that can only be dodged if the player is back to the normal color. This forces the player to constantly switch back and forth.
  • Prototype Goal: Create a color system and spawn system.
  • Features: Fun fast paced game that utilizes color as the main mechanic.
  • Setting/Premise: Block world!
  • Story: Dodge the blocks and survive.
  • Target Audience: For all age groups and genders.
  • Play Time: Anywhere from 30seconds to 30minutes.
  • Strategies: Time the color changes of the player so that it matches the color of the block without moving around too much. Watch out for the black blocks.
  • Prototype Assets: Everything is original except for the main player sprite from OpenArtSource.
  • Prototype Closest Other Game: Reverse bullet hell games?
